# Mouse adult gross anatomy

Description: A structured controlled vocabulary of the adult anatomy of the mouse (Mus).

More information can be found at http://obofoundry.org/ontology/ma

## Versions

### Stable release versions

The latest version of the ontology can always be found at:

http://purl.obolibrary.org/obo/ma.owl

(note this will not show up until the request has been approved by obofoundry.org)

### Editors' version

Editors of this ontology should use the edit version, [src/ontology/ma-edit.owl](src/ontology/ma-edit.owl)
