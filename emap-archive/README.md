# EMAP Archive

The [Mouse gross anatomy and development, timed (EMAP)](https://obofoundry.org/ontology/emap.html) is not only deprecated.

They were downloaded here to the repo from these URL in BioPortal (according to UI uploaded on 02/10/2016), only for long term preservation:

- [RDF XML Format](https://data.bioontology.org/ontologies/EMAP/download?apikey=8b5b7825-538d-40e0-9e9e-5ab9274a9aeb&download_format=rdf)
- [OBO Format](https://data.bioontology.org/ontologies/EMAP/submissions/7/download?apikey=8b5b7825-538d-40e0-9e9e-5ab9274a9aeb)

These are not and will never again be maintained.
